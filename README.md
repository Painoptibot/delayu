# ДелаЮ (newsystem)

Платформа **Дела.ЮГИт** — этапы 0–10 ТЗ реализованы в `delayu/`.

## Запуск

```bat
runserver.bat
```

| URL | Назначение |
|-----|------------|
| http://127.0.0.1:8000/auth/login/ | Вход |
| http://127.0.0.1:8000/ | Главная, статистика |
| http://127.0.0.1:8000/api/v1/cases/ | REST API (M43) |

**Production (Ubuntu 24.04):** см. [docs/deploy-ubuntu-2404.md](docs/deploy-ubuntu-2404.md)

**Учётки:** `admin`/`admin`, `demo`/`demo`, `manager`/`manager`

```bat
.\.venv\Scripts\python manage.py seed_demo
```

## Реализовано

- **M01–M06** — подсистемы, роли, пользователи, ШР, документы, архив
- **M07–M14** — кабинет, сегодня, календарь, канбан, уведомления, лента
- **M22–M26, M05, M30** — дела, реестры, СЭД, журнал, ЭП (демо)
- **M33–M34** — BPM, согласования на карточке дела
- **M37–M38** — чат, комментарии
- **M15–M16, M42–M45, M47–M48, M61, M73, M82–M83, M46** — дашборды, отчёты, интеграции, ИИ, НСИ, мониторинг, лицензии, аудиоархив
- **M43** — JSON API `/api/v1/*`
- **M49–M66, M67–M86** — сервисный слой (`delayu/services/ai.py`), модели данных, лицензирование; UI через админку и расширяемые экраны

Права: матрица **роль × модуль**, меню строится динамически.
