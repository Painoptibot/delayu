# Идентифицирующие материалы — Роспатент

## Файл для загрузки

**`Delayu_ident_materialy.zip`** (~17 Кб, лимит 5 Мб)

Поле формы: **Идентифицирующие материалы** → выберите этот ZIP.

## Содержимое архива

| Файл | Назначение |
|------|------------|
| `00_titul.txt` | Титульный лист |
| `01_referat.txt` | Реферат (900 знаков) |
| `02_src_*.txt` | Фрагменты исходного кода (начало и конец модулей) |
| `02_istochniki_svodka.txt` | Сводка всех фрагментов |

## Пересборка

```bat
.venv\Scripts\python.exe scripts\build_rospatent_ident.py
```

## Важно

- Загружайте **только один** файл (ZIP), если лимит **5 Мб на все файлы** заявки.
- Не добавляйте ZIP вместе с другими тяжёлыми вложениями в одной загрузке.
- Имя файла ≤ 100 символов (текущее имя подходит).
