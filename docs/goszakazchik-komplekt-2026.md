## О продукте «ДелаЮ»

**ДелаЮ** — модульная веб-платформа **ЮГИт** для ведения заявок, учётных и юридических дел, документооборота, сроков и отчётности с разграничением прав доступа. Предназначена для органов власти, муниципальных учреждений и коммерческих организаций; поставляется конфигурациями на едином ядре. Для УЖВ — конфигурация **«АИС УЖВ»** (жилищный учёт, обращения, договоры, жилфонд, отчётность по согласованному ТЗ).

*Слоган:* **Веду дела — ДелаЮ** · *Полное описание для реестра:* `docs/delayu-product-opisanie.md`

**Для извещения (август):**  
*«Внедрение программного комплекса «ДелаЮ» (Дела.ЮГИт), конфигурация «АИС УЖВ», в соответствии с согласованным ТЗ и протоколами визитов 1–4.»*

## 1. Кто за что отвечает

| Роль | Организация | Зона | Контакт по |
|------|-------------|------|------------|
| **Функциональный заказчик** | **УЖВ** | Процессы, поля, экраны, отчёты, пользователи | Визиты 1–3 |
| **Технический заказчик** | **Управление ИКТ (УИКТиС)** | Закупка, сервер, VPN, ИБ, **приёмка по акту** | Сессия 0, визит 4 |

**Правила:**

- Закупку **ведёт ИКТ**; функционал согласует **УЖВ**.
- Сервер при размещении «на стороне администрации» **предоставляет ИКТ** — не обсуждать инфраструктуру только с УЖВ.
- **Приёмку подписывает ИКТ** (комиссия: председатель ИКТ + представитель УЖВ по функционалу).
- Юрлицо в извещении ЕИС (администрация / МКУ «Электронный Краснодар» / иное) — **утвердить до августа**.

## 7. Состав системы

### 7.1. Платформа (общее)

Веб, JWT, роли, журнал аудита, PostgreSQL, Docker, **Astra Linux + Nginx** (по ТЗ), размещение в контуре **ИКТ**.

### 7.2. Модули УЖВ — этап приёмки 02.10

| Подсистема | К 02.10 | Примечание |
|------------|---------|------------|
| Малоимущие + расчёт | Полный цикл | Заявление → заключение |
| Учёт нуждающихся | Реестры, очерёдность | + история, отчёты |
| Молодые семьи | Списки | Законы КК 2704, 2710 |
| Дети-сироты | Карточка дела | Решения Минтруда **вручную** (без I-04, I-08) |
| Договоры | Реестр + шаблоны | docx/pdf |
| Жилфонд | МКД, помещения | Импорт PY-01 |
| Жилконтроль | Проверки, предписания | |
| Переселение | Реестры | Ручной ввод / файлы |
| Обращения | Регистрация, 30 дн. | ОТЧ-5 |
| Отчёты | ОТЧ-1…ОТЧ-10 | Где не требуют I-xx |
| Шаблонизатор | docx/pdf | |
| Интеграции I-xx | **Не входят** | Справочник опций на этап 2 |

**Обмен на этапе 1:** только **файлы** и **PY-01…PY-07** (импорт, миграция, отчёты, ZIP по делу).

## 14. Тезис для руководства (1 абзац)

*По согласованному упрощённому ТЗ внедряется программный комплекс **«ДелаЮ»** (ЮГИт) в конфигурации **«АИС УЖВ»** с полным внутренним функционалом учёта, договоров, контроля и отчётности **без подключения внешних систем** на первом этапе. **УЖВ** задаёт процессы и согласует интерфейсы; **ИКТ** обеспечивает сервер, закупку и приёмку. С мая по июнь — прототип и четыре сессии; в июле — реестр отечественного ПО; в августе — закупка; в сентябре–октябре — внедрение и приёмка **до 02.10.2026**. Интеграции ГИС ЖКХ, СМЭВ и др. — отдельный этап 2.*
